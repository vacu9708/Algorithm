# [226. Invert Binary Tree](226. Invert Binary Tree)

Changing the address a child pointer has moves all its subnodes.
~~~python
def invertTree(self, root):
    def recursion(node):
        if node:
            node.left, node.right = recursion(node.right), recursion(node.left)
            return node
    recursion(root)
    return root
~~~
